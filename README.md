# LLM Reasoning Hint Analysis

**Do hints influence reasoning models?**

A lightweight evaluation of prompt sensitivity on open-weight language models. Each of 10 classic reasoning problems is tested under three prompt conditions: clean, helpful-hint, and misleading-hint. The goal is to measure how much a subtle cue can shift model behaviour — even when the cue is wrong.

---

## Project Goal

Investigate whether subtle hints influence the reasoning behaviour of open-weight language models, and quantify how often misleading hints cause a previously correct answer to become incorrect.

---

## Method

10 reasoning questions spanning arithmetic traps, lateral thinking, and trick questions. Each question appears in three prompt variants:

| Condition | Description |
|-----------|-------------|
| `clean` | Question only, no hint |
| `helpful` | Question + a hint that clarifies the key concept |
| `misleading` | Question + a hint that nudges toward a wrong approach |

Model responses are collected into a dataframe and manually graded against ground-truth answers.

**Model used:** `microsoft/Phi-3-mini-4k-instruct` (no token required, runs on Colab T4)

> To replicate with Gemma 2: set `MODEL_ID = "google/gemma-2-2b-it"`, accept the license on Hugging Face, and add your `HF_TOKEN` to Colab Secrets.

## How to Run

1. Open `reasoning_experiment.ipynb` in [Google Colab](https://colab.research.google.com/)
2. Set runtime to **T4 GPU** (Runtime → Change runtime type)
3. Run all cells in order
4. After the model finishes, manually grade each response in the grading cell
5. Re-run the analysis cell to get accuracy stats and save `results.csv`

---

## Relevance

This project explores **prompt sensitivity** and **reasoning reliability** in LLMs — topics closely related to mechanistic interpretability, LLM evaluation, and adversarial prompt research. The three-condition design mirrors methodology used in cognitive bias studies applied to language models.

---

## Questions Tested

1. "All but 9 die" — language parsing trap  
2. Water jug problem — multi-step planning  
3. Bat and ball — CRT classic  
4. Race overtake — positional reasoning  
5. Months with 28 days — universal set trap  
6. Pill timing — interval vs count confusion  
7. Apple ownership — subject-focus trap  
8. Two fathers two sons — overlapping roles  
9. Rooster egg — false premise  
10. Bus driver name — misdirection by arithmetic

## Result
60% of answers changed when misleading hints were introduced (6/10 questions).
